# copra-gym

## Android WebView app icon

لتغيير شعار تطبيق الأندرويد WebView بصيغة PNG استبدل الملف:

`android-webview-app/app/src/main/res/drawable-nodpi/app_logo.png`

وسيتم استخدامه تلقائيًا كأيقونة التطبيق العادية والدائرية، وكذلك داخل الـ adaptive icon أثناء البناء.

يفضّل أن تكون الصورة مربعة وواضحة (مثل 432×432 بكسل أو أكبر) للحصول على أفضل جودة بعد تصغيرها إلى أحجام أيقونات أندرويد المختلفة.
